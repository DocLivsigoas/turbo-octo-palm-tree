# Ecommerce Store

## Описание проекта
Ecommerce Store - это интернет-магазин, который предоставляет пользователям возможность просматривать товары, добавлять их в корзину и оформлять заказы. Также предусмотрена админ-панель для управления товарами и пользователями.

## Структура проекта
Проект состоит из клиентской и серверной частей:

- **Клиентская часть** (src/client):
  - Реализована на React с использованием TypeScript.
  - Включает страницы для отображения товаров, корзины и личного кабинета пользователя.
  - Содержит компоненты для отображения заголовка, карточек товаров и корзины.
  - Использует контекст для управления состоянием авторизации.

- **Серверная часть** (src/server):
  - Реализована на Node.js с использованием Express.
  - Обрабатывает запросы на авторизацию, управление товарами и корзиной.
  - Включает контроллеры, модели и маршруты для обработки данных.
  - Поддерживает отправку электронных писем для восстановления пароля.

## Установка
1. Клонируйте репозиторий:
   ```
   git clone <URL>
   ```
2. Перейдите в директорию проекта:
   ```
   cd ecommerce-store
   ```
3. Установите зависимости для клиентской части:
   ```
   cd src/client
   npm install
   ```
4. Установите зависимости для серверной части:
   ```
   cd ../server
   npm install
   ```
5. Настройте переменные окружения, скопировав `.env.example` в `.env` и заполнив необходимые значения.

## Запуск
1. Для запуска клиентской части:
   ```
   cd src/client
   npm start
   ```
2. Для запуска серверной части:
   ```
   cd src/server
   npm start
   ```
